# FLUKA_HKS

## Overview

This repository contains FLUKA simulation files for radiation calculations
for the JLab Hall C experiment E12-24-003 (Pb target).

## Contents

- FLUKA input files (.inp)
- No pre-configured flair project file is included.

## How to Use

1. Create a new project in flair.
2. Import the provided FLUKA input file (.inp).
3. Run the simulation within flair or using FLUKA directly.

## Requirements

- FLUKA
- flair
